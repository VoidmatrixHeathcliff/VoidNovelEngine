## 开发快照版本

### VoidNovelEngine-0.1.0-dev.1

#### 新增

+ 项目初始化：VoidNovelEngine视觉小说引擎
+ 实现基础流程图编辑器和运行时框架
+ 实现预览、监控、控制台等多视图基础框架
+ 实现常用演出功能节点和数据逻辑等诸多节点

### VoidNovelEngine-0.1.0-dev.2

#### 新增

+ 新增TIFF、WEBP、和AVIF图片格式支持
+ 新增可自定义的编辑器界面缩放比例
+ 新增调试窗口帧率可见性修改功能
+ 重定向流程脚本目录并更改文件扩展名

#### 修复

+ 修复JPG格式图片加载异常的问题
+ 修复.JPGE扩展名图片加载时忽略的问题
+ 修复无扩展名文件加载时导致引擎启动崩溃的问题
+ 修复显示宽银幕遮幅节点高度二次设置无效的问题
+ 修复显示字幕和对话框节点文本为空时引擎崩溃的问题
+ 修复显示字幕节点颜色范围设置不正确时引擎崩溃的问题
+ 修复部分节点引脚未正确连接时异常未能成功捕获的问题
+ 修复自动保存时节点位置异常导致流程图无法正常打开的问题

#### 优化

+ 优化日志内容
+ 优化颜色值节点图标
+ 优化菜单栏图标和文本内容
+ 优化编辑器内部窗口标题内容
+ 优化编辑器与调试窗口标题内容
+ 优化编辑器与调试窗口默认大小以适应低分辨率显示器
+ 优化纹理采样模式支持更平滑的缩放效果
+ 优化引擎报错时信息提示，将堆栈信息自动写入剪切板以方便反馈
